## 构建项目

``` bash
# 将项目克隆到本地
复制 git clone https://github.com/jdfdcc/n22-phone-ui.git

# 安装项目依赖性
npm install

# 启动项目 
npm run dev

# 编译打包项目
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
